<Installer>
=== средство скачивания и установки пакетов мингв ===
-- описание --
mingw-get - средство помощи пользователям в управлении установками ПО MinGW и MSYS
сконфигурированная как онлайновая база данных xml, которая управляется сопровождающими проэкта
>> choice - выбор
>> offer - предложение 
>> facility - объект, установка, возможность
>> facilitate - способствовать, облегчать
он предлагает выбор между графическим интерфейсом и интерфейсом командной строки, облегчая выборочое скачивание и установку пакетов, предоставляемых проэктом

xml-датабаза предоставляется с механизмом для спецификации интер-пакетных зависимостей
>> permit - разрешение, пропускать
это позволяет конфигурировать любй пакет, так что конечный пользователь может выбрать только тот пакет, который ему нужен конечным образом.
>> ensure - обеспечивать, гарантировать
>> satisfy - удовлетворять, соответствовать, выполнять
избегая mingw-get, чтобы гарантировать, что все зависимости будут автоматически соответствовать во время установки

-- лицензионные положения --
mingw-get - свободное ПО. Разрешено копировать, модифицировать и распространять это ПО под имеющейся GNU GPL v3 (или более поздней) лицензией, опубликованной FSF.
смотри файл COPYING, который распространяется с пакетом mingw-lic-package и с исходным кодом для лицензирования деталей

важно, в частности, что mingw предоставляется как есть, надеясь, что его польза может быть подтверждена, но БЕЗ ГАРАНТИЙ В ЛЮБМ ВИДЕ
>> merchantability - товарный
ни встроенной товарной гарантии, ни о пригодности, ни дл якаких других целей
>> circumstance - обстоятельство, случай
>> liable - подлежащий, ответственный
ни прикаких обстоятельствах автор или проэкт mingw не несет ответственности за урон, полученный от использования этого ПО

<MinGW/Base>
пакеты в директории MinGW/Base поддерживаются командой разработчиков MinGW
они предоставляют основные средства, требуемые для базовойй установки MinGW

Для установки файлов, распространяемых MinGW'ом, мы предлагаем использовать mingw-get-installer: https://sourceforge.net/projects/mingw/files/Installer
>> suggest - предлагать 
...
---------------------------------------------------------------------
<MinGW/Extension>
пакеты в директории MinGW/Extension поддерживаются командой разработчиков MinGW
они предоставляют полезные средства, но они не рассматриваются как необходимые для базовой установки MinGW
...
---------------------------------------------------------------------
<MinGW/Contributed>
пакеты в директории MinGW/Contributed поддерживаются индивидуальными разработчиками, а не командой разработчиков MinGW
Здесь часто полезные дополнения для установки MinGW, но они не рассматриваются как необходимые
