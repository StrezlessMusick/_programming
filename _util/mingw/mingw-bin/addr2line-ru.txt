Usage: addr2line [option(s)] [addr(s)]
 Конвертирует адреса в пары номер строки/имя файла.
 Если адреса не заданы в командной строке, будет читать stdin
 The options are:
  @<file>                читает опции из <file>
  -a --addresses         показывает адреса
  -b --target=<bfdname>  устанавливает бинарный формат файла
  -e --exe=<executable>  Устанавливает имя входного файла (default is a.out)
  -i --inlines           Раскрывает inlined функции
  -j --section=<name>    читает смещения относительно секции взамен адресов
  -p --pretty-print      делает вывод более читабельным для людей
  -s --basenames         удаляет имена директорий
  -f --functions         показывает имена функций
  -C --demangle[=style]  восстанавливает имена функций
  -h --help              Display this information
  -v --version           Display the program's version

Usage: mingw32-addr2line [option(s)] [addr(s)]
 Convert addresses into line number/file name pairs.
 If no addresses are specified on the command line, they will be read from stdin
 The options are:
  @<file>                Read options from <file>
  -a --addresses         Show addresses
  -b --target=<bfdname>  Set the binary file format
  -e --exe=<executable>  Set the input file name (default is a.out)
  -i --inlines           Unwind inlined functions
  -j --section=<name>    Read section-relative offsets instead of addresses
  -p --pretty-print      Make the output easier to read for humans
  -s --basenames         Strip directory names
  -f --functions         Show function names
  -C --demangle[=style]  Demangle function names
  -h --help              Display this information
  -v --version           Display the program's version

mingw32-addr2line: supported targets: pe-i386 pei-i386 elf32-i386 elf32-little elf32-big plugin srec symbolsrec verilog tekhex binary ihex
Report bugs to <http://www.sourceware.org/bugzilla/>
