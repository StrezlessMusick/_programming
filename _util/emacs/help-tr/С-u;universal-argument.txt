C-u runs the command universal-argument, which is an interactive
compiled Lisp function in `simple.el'.

It is bound to C-u.

(universal-argument)

Начинает числовой аргумент для следующей команды.
Цифры или знак минуса, следующие за C-u задают числовой аргумент.
C-u следующий за цифрами или знаком минуса завершает аргумент.
C-u без цифр изи знака минуса предоставляет 4 как аргумент.
Повтор C-u без цифр или знака минуса
 умножает аргумент на 4 каждый раз.
Для некоторых команд, только C-u само по себе является флагом
which is different in effect from any particular numeric argument.
Эти команды включают C-@ и M-x start-kbd-macro.
