LOCALE(1)

ИМЯ
	выдает информацию касательно локали

СИНТАКСИС
	locale [-amsuUvh]
	locale [-ck] NAME

ОПЦИИ
	-a, --all-locales
		перечисляет доступные поддерживаемые локали
	-c, --categoty-name
		перечисляет информацию о данной NAME катеогрии
	-k, --keyword-name
		перечисляет информацию о данном NAME ключевом слове
	-m, --charmaps
		перечисляет доступные таблицы символов
	-s, --system
		выводит системную локаль по умолчанию
	-u, --user
		выводит локаль пользователя по умолчанию
	-U, --utf
		добавляет ".UTF-8" к результатам
	-v, --verbose
		больше вывода
	-h, --help
		этот текст

ОПИСАНИЕ
	locale без параметров печатает информацию о настройках окружения текущей локали

	Опция -u печатает пользовательскую виндовую локаль.
	Опция -s печатает вместо этого системную локаль по умолчанию.
