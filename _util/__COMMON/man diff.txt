diff - сравнивает файлы построчно

diff [options]... files

обязательные аргументы для длинных опций являются обязательными также и для коротких.
--normal
	выводит нормальную разницу (по умолчанию)
-q, --breaf
	сообщать, только если файлы различаются
-s, --report-identical-files
	сообщать, когда 2 файла совпадают
-c, -C num, --context[=num]
	
-l, --paginate
	пропускает вывод через pr для нумерования страниц


-a, --text
	обрабатывать все файлы как текст
