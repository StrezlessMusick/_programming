TERMINALS(7)

НАЗВАНИЕ
terminals - Список поддерживающихся терминалов.

ОПИСАНИЕ
Следующий список, выведенный из файла /etc/termcap, показывает имена терминалов (пригодных для использования переменной оболочки TERM) и короткое описание терминала. Консультация в termcap(7) будет помогать пользователям при создании вводов termcaр для терминалов, не поддерживающихся в настоящее время.
