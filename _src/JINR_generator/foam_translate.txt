Abstract
Описывается общего назначения, самоадаптирующийся Monte Carlo (MC) генератор событий (симулятор) Foam.
>> Varance - дисперсия
>> mean - означать
>> means - средства
Высокая эффективность MC, that is маленький максимальный вес дисперсия веса MC достигается 
при помощи средств деления домена интеграции на маленькие клетки.
>> simplex(ces) - симплекс(ы) - многомерные тетраэдры
>> certain - определенный
Клетки могут быть n-мерными симплексами, гиперпрямоугольниками или определенными их продуктами.
Сетка клеток, называемая “foam”, производится в процесся бинарного деления клеток.
>> choice - выбор
>> drive
>> ratio - соотношение
>> average - средний
>> ! variance - дсперсия
Выбор следующей клетки для деления и позицию/направление гиперплоскости деления ведется алгоритмом
который оптимизирует соотношения максимального веса к среднему весу или (опционально) общей дисперсии.
>> arbitrary - произвольный
Алгоритм в состоянии иметь дело, в принципе, с произвольным расположением сингулярностей в распределении.
Как люой МК генератор, Foam может также использоваться для МК интегрирования.
>> perform - выполнять
С типичным CPU персонального компьютера, программа в состоянии выполнять адаптивную интегрирование/симуляцию
для относительно маленького числа размерностей (≤ 16).
>> inevitably - неизбежно
Всвязи с продолжающимся прогрессом мощностей ЦП, это ограничение будет неизбежно уменьшаться.
>> aim - цель
Программа нацелена (и уже протестирована) как компонент МК генераторов для кспериментов физики высоких энергий.
>> few - несколько, мало
Показано несколько примеров соответствующихприложенийц.
Код ФОАМа написан в полностью объектно-ориентированном стиле, на the C++.
>> slightly - чуть, слегка
Две другие версии, написанные с чуть уменьшенной функциональностью, доступны на Fortran77.
Исходный код доступен по адресу http://jadach.home.cern.ch/jadach/.
