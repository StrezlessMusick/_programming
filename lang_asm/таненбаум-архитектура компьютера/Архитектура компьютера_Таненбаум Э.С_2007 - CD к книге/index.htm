This CD contains two simulators for use with Structured Computer Organization, Fifth Edition, by Andrew S. Tanenbaum. The simulators are:

8088 Assembler and Tracer Toolkit (for Intel 8088)
This software can be installed on the Windows, Linux and Solaris platforms.
Mic-1 MMV Simulation Environment (for the Mic-1 machine discussed in Chapter 4)
This software can be installed on the Windows, Linux and Macintosh platforms.

NOTE: Each software package can be installed and run independently of the other. You may install one or both.

Please review the System Requirements for this CD prior to installing any software.
