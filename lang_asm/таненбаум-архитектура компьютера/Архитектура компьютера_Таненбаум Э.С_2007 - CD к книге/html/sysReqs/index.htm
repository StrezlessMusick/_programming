System Requirements

Here are the Minimum System Requirements to run the applications supplied on this CD.

To use the software supplied on this CD, you need:
- A Pentium II 450-MHz (or faster) processor
- One of the following operating systems:

Microsoft® Windows®

  • Server 2003
  • XP Professional
  • XP Home Edition
  • XP Media Center Edition
  • XP Tablet PC Edition
  • 2000 Professional (SP3 or later required for installation)
  • 2000 Server (SP3 or later required for installation)
  • Windows ME
  • Windows 98 (1st or 2nd Editions)

Intel/Linux (e.g., little endian) platforms

  • Red Hat 9.0

Macintosh

  • Mac OS X 10.3

SPARC/Solaris (e.g., big endian) platforms

  • Solaris 10

You also need:

  • 10 MB of available hard-disk space
  • CD-ROM drive
  • Internet Browser (Internet Explorer 6.0, Firefox 1.0, Mozilla 1.7.x, or Safari 1.2)
  • Internet Connection - required only for downloading additional software, such as updates
  • Screen Resolution 800 x 600 (minimum)
  • Sun's Java™ JRE 1.4 (Only required if the Mic-1 software is installed)
  • Zip decompression utility (Only required if you will be modifying and recompiling the source files of the Mic-1 software)

You must, of course, also meet your operating system's minimum hardware requirements.

Beyond the minimum amount of RAM required by your operating system, there are no additional RAM requirements to run this CD-ROM.
