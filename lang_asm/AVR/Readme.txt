Впервые - подробное описание микроконтроллера ATtiny 2313 на русском языке.

А.В.Белов.
Микроконтроллеры AVR в радиолюбительской практике.

Данная книга представляет собой справочник, в котором представлено полное и подробное описание одной конкретной микросхемы – микроконтроллера ATtiny2313 семейства AVR фирмы Atmel. Описание построено на основе оригинальной технической документации на микросхему и содержит описание всех регистров, всех видов памяти и всех внутренних систем микроконтроллера.



Данный файл представлен исключительно в ознакомительных целях и способствуют быстрейшему профессиональному росту читателей .Публикация данного документа не преследует никакой коммерческой выгоды.
Все авторские права на данный файл сохраняются за правообладателем.
