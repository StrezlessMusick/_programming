#NoTrayIcon Директива, указывающая на то, что индикатор AutoIt не должен отображаться при запуске сценария. 
#comments-start Директива начала многострочного комментария. 
#comments-end Директива конца многострочного комментария. 
#include Директива включения файла в текущий скрипт. 
#include-once Директива однократного включения файла в текущий скрипт. 

Dim / Global / Local / Const Объявление переменной, константы или создание массива. 
ReDim Изменить размер имеющегося массива. 
False / True Булевы значения, используемые в логических выражениях. 
Enum Перечислить переменные. 

If...Then Условное выполнение одиночного выражения. 
If...ElseIf...Else...EndIf Условное выполнение нескольких выражений. 
Select...Case...EndSelect Условное выполнение выражения в конструкции Select. 
Switch...Case...EndSwitch Условное выполнение выражения в конструкции Switch. 
ContinueCase Прервать текущее выполнение выражения case и возобновить его для следующего выражения case в конструкциях Select или Switch. 

Do...Until Цикл, основанный на значении выражения. 
For...Next Цикл, основанный на значении выражения. 
For..In..Next Цикл, основанный на перечислении всех членов коллекции объектов или элементов массива 
While...WEnd Цикл, основанный на значении выражения. 
ContinueLoop Продолжить цикл While/Do/For. 
ExitLoop Завершение цикла While/Do/For. 
With...Endwith Конструкция, сокращающая ссылку на переменные объектного типа 



Func...Return...EndFunc Функция пользователя. 
Default Ключевое значение, применяемое при вызове функции. 
Func OnAutoItExit ()...EndFunc Функция пользователя, которые будут вызываться при завершении работы сценария. 
Func OnAutoItStart ()...EndFunc Функция пользователя, которые будут вызываться до начала выполнения сценария. 

Exit Завершение сценария.
