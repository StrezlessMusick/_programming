Вложенные конструкции проверки if/then

Проверки условий с использованием конструкции if/then могут быть вложенными. Конечный результат будет совпадать с результатом использования составного оператора сравнения && .

a=3

if [ "$a" -gt 0 ]
then
  if [ "$a" -lt 5 ]
  then
    echo "Значение переменной \"a\" лежит где-то между 0 и 5."
  fi
fi

# Такой же результат, как и в случае:

if [ "$a" -gt 0 ] && [ "$a" -lt 5 ]
then
  echo "Значение переменной \"a\" лежит где-то между 0 и 5."
fi

Пример 34.4, «Using arrays and other miscellaneous trickery to deal four random hands from a deck of cards» демонстрирует использование вложенных проверок условий с конструкцией if/then.
