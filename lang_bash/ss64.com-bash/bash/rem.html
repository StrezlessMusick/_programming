#

A comment or remark is indicated by starting the line with the # character

   # This is a description
   # Another remark
   command
   # Another remark

#Now stand in the place where you work, Now face West
Think about the place where you live, Wonder why you haven't before# ~ REM 'Stand'

Equivalent Windows command: REM - comment
