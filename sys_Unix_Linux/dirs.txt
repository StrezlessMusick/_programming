существует Filesystem Hierarchy Standard, см википедию


/boot	загрузка (файлы загрузчика, ядро, initrd, System.map)

/bin	бинарники (основные утилиты, необходимые всем пользователям, cat, ls, cp)
/sbin	бинарники, которые используются после успешного подключения /usr
/lib	разделяемые библиотеки и модули (драйверы устройств)
/etc	конфиги
/etc/rc.d	загрузки системы
/etc/skel	образец пользователя по умолчанию
/etc/sysconfig	некоторые (но не все) системные
/etc/X11	X11

/proc	файлы процессов
/dev	девайсы

/var	переменные файлы
/var/catman/cat[1..9] готовые маны
/var/run	файлы с PID'ами процессов и текущей инф-ой
/var/spool	файлы программ, ожидающие обслуживание
		(at, cron, lpd(на печать), mail, news, uucp)
/var/adm	учетная и диагностиче кая инфа для сисадмина
/var/log	логи
/var/lock	для резервного копирования управляющих файлов системы
/var/backups	бекапы
/var/tmp	tmp

/usr	все основные общие приложения, и их конфиги
	лучше в отдельный раздел, или на сетевой диск только для чтения
/usr/bin	утилиты и приложения для обычных пользователей
/usr/sbin	программы си темного администрирования, не трбующиеся при загрузке
/usr/X11R6	X11
/usr/lib	динамические и статические библиотеки, и программы которые не вызываются непосредственно
/usr/etc	конфиги
/usr/dict	словарь для спелл-чекеров
/usr/include	для си(++)
/usr/src	исходники
/usr/man	ман в исходном формате <locale>/man[1..9] - 
/usr/tmp	tmp
/usr/local	локально (только на этой машине) установленные программы
		(bin, sbin, lib, etc, src, doc, man, info, )

/root	домашний каталог суперпользователя
/home	домашние каталоги пользователей
/tmp	временные файлы
/media
/mnt	для монтирования
/lost+found	файлы и каталоги, у которых потерялись родительские директории, и нашлись во время работы команды fsck
