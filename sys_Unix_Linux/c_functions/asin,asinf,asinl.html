asin, asinf, asinl – расчет арксинуса

Синтаксис:

#include < math.h >

double asin (double x);
float asinf (float x);
long double asinl (long double x);

Аргументы:

x – число от -1 до 1, арксинус которого требуется рассчитать.

Возвращаемое значение:

При успешном завершении функции возвращают значения арксинуса аргумента.

Если значение аргумента лежит вне диапазона от -1 до 1, то функции возвращают значение NaN, а переменной errno присваивается значение [EDOM]

Описание:

Функции рассчитывают значение арксинуса аргумента X. Значение полученного угла представляется в радианах.

Аргумент и возвращаемое значение функции asin являются числами с плавающей точкой двойной точности (тип double, точность не менее десяти значащих десятичных цифр, разрядность - 64).

Аргумент и возвращаемое значение функции asinf являются числами с плавающей точкой (тип float, точность не менее шести значащих десятичных цифр, разрядность - 32).

Аргумент и возвращаемое значение функции asinl являются числами с плавающей точкой повышенной точности (тип long double, точность не менее десяти значащих десятичных цифр, разрядность - 80).

Пример:

В примере рассчитывается арксинус от 0.43 с помощью функций asin, asinf и asinl, и результат выводится на консоль. Обратите внимание на точность полученных результатов. У арксинуса, рассчитанного с помощью функции asinf, будет самая маленькая точность, а у рассчитанного с помощью функции asinl – самая большая. 


#include < stdio.h >    //Для printf
#include < math.h >   //Для asin, asinf, asinl
                     
int main (void)
{    
   //Вывод аргумента
   printf (“Аргумент: 0.43\n”);
   //Расчет и вывод результата работы функции asinf
   printf (“asinf : %.20f \n”, asinf(0.43) );
   //Расчет и вывод результата работы функции asin
   printf (“asin  : %.20f \n”, asin(0.43) );
   //Расчет и вывод результата работы функции asinl
   printf (“asinl : %.20Lf \n”, asinl(0.43) );
	 
   return 0; 
}

Результат:

Аргумент: 0.43
asinf : 0.44449278712272644043
asin : 0.44449277693581901172
asinl : 0.44449277693581903137
