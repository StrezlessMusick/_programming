Класс функционального объекта "хеш" по умолчанию.
Унарный класс функционального объекта, который определяет хеш-функцию, по умолчанию используемую стандартной библиотекой.

Функциональный вызов возвращает хеш-значение своего аргумента:
Хеш-значение - значение, которое полностью зависит от своего аргумента,
возвращая всегда одинаковое значение для одинаковых аргументов (для данного зыпуска программы).
Возвращенное значение должно иметь маленький шанс совпадать со значением для другого аргумента
(с шансом совпадения, стремящимся к 1/numeric_limits<size_t>::max)

Другие типы функциональных объектов могут использоваться для неупорядоченных контейнеров
предоставляя поведение, как определено выше и они дожны быть по крайней мере копи-конструируемыми и деструктируемыми функциональными объектами.

Хеш по умолчанию это шаблонный класс, который не определен в общем случае.
Но все реализации библиотеки предоставляют по крайней мере следующие специализации
